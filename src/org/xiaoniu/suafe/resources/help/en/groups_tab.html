Groups Tab

Overview

The Groups Tab consists of three sections. The far left list displays all current defined groups. The upper right section displays all members of the currently selected group. The lower right section displays all access rules that reference the currently selected group. Selecting a different group automatically refreshes the two right sections.

Actions

Located group list are the group action buttons. Clicking on Add allows you to add another group. Clicking edit displays the edit screen for all currently selected groups. Clicking delete deletes all currently selected groups.


Below the group members list is the Add/Remove Members action button. This allows you to add and remove members from the selected groups. Double-click a group or user name to add or remove the group or user from the list of members, or highlight and click the appropriate button. You may also change group membership by highlighting the member/non-member and hitting the space bar.

Double-clicking on a group in the group list displays the edit screen for the selected group. Double-clicking on a member name in the member list displays the member. Selected groups and group details are displayed in the Groups tab and users and user details are displayed in the Users Tab.

Considerations

Groups with no members and groups that are not referenced are persisted.

Deleting a group also deletes all access rules that reference that group. Group members are not deleted.
