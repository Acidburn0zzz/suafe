Welcome to Suafe

Suafe is a "Subverion User Authentication File Editor" designed to ease the headache of maintaining authz files by hand. It is a graphical application written using Java and Swing and is available for distribution and use as detailed in the GNU General Public License.

The latest version of Suafe is always available at the Suafe home: http://suafe.xiaoniu.org

Subversion browsing capabilities are compliments of the JavaSVN libary, version 1.0.6 (http://tmate.org/svn/). Complete source code for Suafe is available at http://suafe.xiaoniu.org.
